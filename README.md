# Advent of code 2025

This repo collects the solutions for 2025's advent of code. 
You can find the challanges [here](https://adventofcode.com).

The repo is structured as follows:
```
problem-1/
    first-star/
        solution.c
    second-star/
        solution.c

problem-2/
    first-star/
        solution.c
    second-star/
        solution.c

problem-(problem_number)/
    (star_number)-star/
        solution.*
    ...

```

Feel free to add your own solutions alongside mine! **Any language is welcome!**
